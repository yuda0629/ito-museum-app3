# 伊都国遺跡デジタルアーカイブ

## ■ 概要
文化財情報はテキストベースで管理されることが多く、空間的な関係性の理解が困難であるという課題がある。
本アプリで伊都国を対象地域として選定したのは、現在居住する福岡の地元遺跡を扱いたいという思いと、将来的に博物館学芸員として地域文化財に携わることを目指しているためである。伊都国は弥生時代を代表する遺跡群が集中する地域であり、GISを用いた空間的な可視化によって、その分布の意義を広く伝えられると考えた。

## ■ 解決アプローチ
本アプリではGIS（地理情報）を活用し、
遺跡を地図上に可視化することで以下を実現した。

- 時代ごとの分布把握
- 種別ごとの比較
- 空間的理解の促進

## ■ 機能
- 地図表示（Leaflet）
- 時代・種別フィルタ
- クリックによる詳細表示（展示カード）
- 時代別カラー分類（凡例付き）

## ■ 技術構成
- R / Shiny
- Leaflet
- データクレンジング（CSV処理）

## ■ 工夫点
