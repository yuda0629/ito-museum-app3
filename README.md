# 伊都国遺跡デジタルアーカイブ

## ■ 概要
本アプリは伊都国地域の遺跡情報を地理情報として可視化し、
時代および種別ごとの分布を直感的に理解できるよう設計したものである。

## ■ 動機
福岡在住であり、伊都国は邪馬台国論争とも関わる重要な遺跡群であることから選定した。

## ■ 背景
文化財情報はテキストベースで管理されることが多く、
空間的な関係性の理解が困難であるという課題がある。

## ■ 解決アプローチ
本アプリではGIS（地理情報）を活用し、
遺跡を地図上に可視化することで以下を実現した。

- 時代ごとの分布把握
- 種別ごとの比較
- 空間的理解の促進

## ■ 機能
- 地図表示（Leaflet）
- 時代・種別フィルタ
- クリックによる詳細表示（展示カード）
- 時代別カラー分類（凡例付き）

## ■ 技術構成
- R / Shiny
- Leaflet
- データクレンジング（CSV処理）

## ■ 工夫点
- 列名ゆらぎ・文字コード対応
- 欠損データ除去
- UIを博物館展示風に設計

## ■ 今後の展望
- デジタルアーカイブ連携
- 画像・3Dデータ統合
- 教育コンテンツへの応用

## ■ 想定利用者
- 博物館学芸員
- 文化財研究者
- 一般来館者

## ■ ソース
- データ出典：伊都国歴史博物館公式サイト、
- 福岡県教育委員会公開資料、
- 糸島市文化財課公開情報等をもとに独自に整理・構造化
